Программа по моделированию игры "Жизнь" на Python.

Для отрисовки игры была использована библиотека pygame.
Реализован фрактальный и волновой автоматы, игра "Жизнь".

Код программы в main.py, есть .exe файл - в папке dist/main/main.exe

Управление в автомате "Жизнь":
"1" - нарисовать 4 периодические фигуры
"2" - нарисовать глайдер
"3" - нарисовать периодическую фигуру с периодом 8
"Space" - перейти к следующему поколению

-----------------------------

A program to simulate the game "Life" in Python.

The library pygame was used to render the game.
Implemented fractal and wave automata, the Game of Life.

The program code is in main.py, there is an .exe file - in the dist/main/main.exe folder

Controls n the Game of Life:
"1" - draw 4 periodic shapes
"2" - draw a glider
"3" - to draw a periodic figure with period 8
"Space" - pass to the next generation
